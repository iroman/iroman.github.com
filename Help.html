    Also it contains a brief description of how Roman numerals work, and some basic rules used in Roman numerals elaboration.


    We didn’t include extensive material about Roman numerals system because that is not our goal, if you are really interested in how Roman numerals work, and  their common uses, you can visit any of the Helpful Links we provide here, where you can find very well documented information about the theme.

   

    The following are some basic rules, that are also included in Help, we followed in iRoman implementation, that can be used to understand Roman numerals.


    A symbol can't be repeated more than three times, e.g. "IIII" is an invalid Roman numeral representation for the number four. However the four, represented as "IV", was originally represented by romans as "IIII", but in modern times this notation is not common.


     A symbol can be repeated only if it is a ten-power. For instance "C" (100), can be repeated, though "D" (500) can't.


    A larger symbol, can't be placed after a smaller one. For example, "X" (10) can't be placed after "V" (5). However there are exceptions for this rule: the two-characters symbols, e.g. "IX" (9), "IV" (4), and so on.


    This is a complex one. After a two-character symbol, such as "IV" (4) or "XL" (40), can only be placed symbols that are smaller than the first of the two-character symbol. For example, after "IV" (4) nothing can be place 'cause there is no symbol smaller than "I" (1). However after "XL" (40) can appear "V" (5) or "I" (1) which are the symbols smaller than "X" (10).
