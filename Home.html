    iRoman is an utility that allows you to convert from Roman numerals to Arabic numerals, and vice-versa. When converting, it displays in a different way, the symbols, or numbers that will result in an invalid numeral.


    Also when used in professional mode, iRoman provides error correction, displaying an explanatory message with the error cause.


    iRoman  can be used in simple mode, where you can only convert Roman numerals from 1 to 3,999, this is the default mode, when the screen is vertically oriented.

   

    Also can be used in professional mode where you can convert Roman numerals from 1 to 3,999,999, when your screen is horizontally oriented, and allows you to convert from, and to, Roman numerals without changing your current view. Also provides error correction displaying an explanatory message with the cause of the error.

Summary

iRoman features

  1. -Easy to use interface

  2. -Two-way Roman numerals conversion

  3. -Error correction, with user friendly explanatory messages

  4. -Built-in help with the fundamentals about Roman numerals
