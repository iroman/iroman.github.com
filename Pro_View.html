    iRoman Pro View is the most powerful feature of iRoman. It can be used only when screen is horizontally oriented. It provides in a single view two-way Roman numerals conversion method. If the user wants to know the Roman representation of an Arabic number he only needs to press the numbers he wants to convert; on the other hand if the user wants the Arabic representation of a Roman numeral, he only needs to press the symbols he wants to convert.


    iRoman Pro View can convert numbers as big as 3,999,999. As in Simple View, when pressing a number or a symbol can result in an invalid numeral, that symbol or number is painted in a different, like blurred, style. And you can simply erase the last entered symbol by pressing the arrow on the right.


    But, how can the user enter a large Roman numeral, lets say 200,000? Just press the arrow on the left, this arrow makes the function of shift, multiplying by 1,000 the symbol value, then just press the “C” twice and that’s it.

   

    Another powerful feature of Pro View is error correction. If the user, tries an invalid symbol combination an explanatory easy-to-read message is shown in the bottom of the screen explaining the cause of the error.
